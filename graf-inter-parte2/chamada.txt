Quando finalizamos as etapas de formatação e manipulação dos dados, podemos criar estatísticas e visualizações de dados a partir dos mesmos. Quando o assunto é análise descritiva, podemos utilizar a Visualização de Dados para entender o comportamento dos dados e idenficar padrões. Entretanto, muitos pacotes geram gráficos estáticos ou inanimados, o que pode decepcionar alguns usuários. Dessa forma, o post de autoria do PETiano João Matheus Hneda, tem como objetivo apresentar alguns conceitos envolvendo Visualização de Dados e mostrar algumas bibliotecas do R que auxiliam na confecção de gráficos interativos.
